## 目录介绍
* 除了getData目录下的文件，其他的都是小程序的代码
* getData目录下的代码为后台代码，即提供服务API的
## 启动
* 切换到目录`cd getData/src` 后台 `npm i` 下载依赖, `node index.js` 
* 小程序修改一下 hotDetail页面里的js中的后台请求API就ok。
